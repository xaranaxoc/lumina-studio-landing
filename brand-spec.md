# Lumina Studio — система лендинга

Источник: бренд-спек встроен в бриф пользователя.

## Палитра
- `--bg`: `#ffffff` (белый холст)
- `--surface`: `#f5f7fa` (светло-серый блок)
- `--fg`: `#0a0b0d` (почти чёрный)
- `--muted`: `#5b6472` (приглушённый серо-синий)
- `--border`: `#e6e9ef` (светлая граница)
- `--accent`: `#3b82f6` (фирменный синий из брифа)
- `--accent-ink`: `#2563eb` (тёмнее — для текста/контраста ≥4.5:1)

## Шрифты
- Дисплей: `'Unbounded'` (геометрический, кириллица) — заголовки, лого
- Тело: `'Manrope'` (чистый гротеск, кириллица) — текст, UI
- Пара: Unbounded display + Manrope body

## Постура
- Белый фон по умолчанию, тёмный текст, один синий акцент
- Крупные фотографии — главный визуальный носитель (кейсы, hero)
- Светящаяся буква L — единственный «огонёк» (decisive flourish), синий glow
- Минимализм: воздух, тонкие границы, мягкие тени только у фото/плавающих карточек
- Скругления: 16–20px у карточек, кнопки pill / 10px
- Акцент-бюджет: ≤2 видимых использования `--accent` на экран
